
SOURCE jQueryUI Themes

Drop here jQueryUI Themes from:
http://jqueryui.com/themeroller/#themeGallery

1. Download a theme from JQueryUI

2. Create a new folder in etc/visual/ui/themes/my-theme

3. Copy:
	jquery-ui-custom-theme/css/custom-theme/jquery-ui.css
	to:
	etc/visual/ui/themes/my-theme/jquery-ui.css

4. Copy:
	jquery-ui-custom-theme/css/custom-theme/images/*
	to:
	etc/visual/ui/themes/my-theme/images/*

5. Test it before use ...
